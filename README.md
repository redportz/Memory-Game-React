# Memory Game (React)

A responsive memory card game built with React that supports difficulty settings and optional multiplayer.

## Live Demo
https://redz-memory-game.netlify.app

## Features
- Multiple card caegories
- Diffficulty selection
- Single-player and multip,layer modes
- Win detection and game reset
- Responsive UI
- Deployed on Netlify

## Tech Stack
- React
- JavaScript (ES6+)
- CSS
- Netlify

## What I Learned
- Managing complex states across components
- Conditional rendering and game logic
- Component-driven UI design
- Deployment and building troubleshooting
